# LAB_PRATIQUE_SECURISATION_RESEAU_AVEC_FORTIGATE
Ce dépôt présente un laboratoire pratique de sécurité réseau réalisé dans un environnement virtualisé. L’objectif principal de ce lab est de mettre en œuvre un pare-feu FortiGate et d’explorer ses fonctionnalités essentielles dans un contexte proche d’un environnement d’entreprise.
